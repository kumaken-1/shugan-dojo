# 主眼道場

**書く前に、見抜く。** 1日1分の診断遊びから始めて、いつのまにか主眼が書けるようになる、小学校教師向けの練習アプリです。

👉 **アプリを開く: https://kumaken-1.github.io/shugan-dojo/**

## できること

| モード | 時間 | 内容 |
|---|---|---|
| 今日の一問 | 1分 | 毎日1問、弱い主眼を診断（全員同じ問題。職員室で答え合わせをどうぞ） |
| 修理工房 | 5分 | 弱い主眼を部品単位で直す練習（A修理・動詞修理・ひっかかり修理） |
| 執筆室 | 15分 | B→A→ひっかかり→プロセスの4箱を埋めて一文を組み立て、7項目セルフチェック→指導案へコピー |
| 記録 | - | 段位・連続日数・七つの目の伸びしろ |

## 主眼とは

「この子どもたちが（A）、何にひっかかり、どう考えて、授業のあとに何をすることができるのか（B）」を一文で言い切ったもの。

基本型: **「〜という子どもたちが、〜という授業場面で、〜を〜することを通して、〜することができる。」**

## 安心して使えます

- 記録は端末内（ブラウザのlocalStorage）にのみ保存。**外部送信は一切ありません**
- 児童の個人情報は扱いません
- サーバー不要。`index.html` をダウンロードしてダブルクリックでも動きます

## 道場訓

> むずかしいことをやさしく、やさしいことをふかく、ふかいことをおもしろく、おもしろいことをまじめに、まじめなことをゆかいに、そしてゆかいなことはあくまでゆかいに
